# System-BookRental
책 대여를 위한 시스템으로 Oracle과 C#을 사용하여 만든 프로그램

## 사용
1. Oracle DataBase
2. C# Programming
3. [bunifuframework](https://bunifuframework.com/) // 유료프로그램 (Modal Windows의 Round ratio를 결정하기 위해 사용)

## 기능

- 도서검색
- 도서예약
- 인기도서
- 도서후기
- 대출내역
- 회원정보
- 대여/반납
- (도서)등록/폐기
- 통계
- 회원관리
